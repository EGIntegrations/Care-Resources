# Care Resources App Documentation

This is the complete client handoff documentation website for the **Care Resources** mobile application.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ LTS
- npm (comes with Node.js)

### Installation & Setup
```bash
# Clone the repository
git clone [repository-url]
cd care-resources-docs

# Install dependencies
npm install

# Start development server
npm run start
```

The site will open at `http://localhost:3000` and go directly to the documentation.

## 📱 About the Care Resources App

**Care Resources** is a production-ready React Native mobile application featuring:

- Multi-layer authentication (Google OAuth, biometric, SSO)
- Secure video content hub
- Interactive care pathways with FAQ system
- Biometrically-protected contact directory
- Community features and event management
- AWS serverless backend infrastructure

### Technical Details
- **Framework**: React Native with Expo SDK 53
- **Language**: TypeScript (strict mode)
- **Backend**: AWS (DynamoDB, Lambda, API Gateway, S3, CloudFront)
- **Bundle ID**: `com.careresources.app`
- **Platforms**: iOS and Android
